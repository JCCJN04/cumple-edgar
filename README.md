# 🎉 Invitación Digital — Cumpleaños 50 de Edgar

Invitación digital interactiva para la celebración del 50° cumpleaños de Edgar. Diseñada para compartirse por WhatsApp o cualquier red social mediante un link directo.

## ✨ Características

- **Diseño elegante** en negro y dorado con tipografías premium (*Cormorant Garamond* y *Great Vibes*)
- **Cuenta regresiva en tiempo real** con días, horas, minutos y segundos
- **Música de fondo** — *Bruno Mars: I Just Might* con reproducción en loop
- **Botón de silenciar/activar** música flotante
- **Enlace directo a Google Maps** con la ubicación del evento
- **Efectos visuales** — bokeh dorado animado y destellos
- **100% responsive** — optimizado para móviles y escritorio

## 🚀 Correr localmente

**Requisitos:** Node.js 18+

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

La invitación estará disponible en `http://localhost:3000`

## 🎵 Audio

La canción (*Bruno Mars — I Just Might*) debe estar en `public/cancion.mp3`. No se incluye en el repositorio por derechos de autor — descárgala con el script incluido:

```bash
python descargar_audio.py
```

## 🌐 Deploy

El proyecto se despliega en [Vercel](https://vercel.com) y se integra al dominio principal `invitacionesdigitales.com.mx` mediante rewrites configurados en el proyecto principal.
